# 345. Reverse Vowels of a String

**Difficulty:** Easy

## Topics

- Two Pointers
- String

## Problem Statement

Given a string s, reverse only all the vowels in the string and return it. The vowels are 'a', 'e', 'i', 'o', and 'u', and they can appear in both lower and upper cases, more than once. Input: s = "IceCreAm" Output: "AceCreIm" Explanation: The vowels in s are ['I', 'e', 'e', 'A']. On reversing the vowels, s becomes "AceCreIm". Input: s = "leetcode" Output: "leotcede" 1 <= s.length <= 3 * 105 s consist of printable ASCII characters.
